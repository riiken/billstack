# Changelog

All notable changes to BillStack will be documented in this file.

## [1.1.0] - 2025-01-XX

### Added
- **User Feedback System**: New prominent feedback button in the header for easy access
- Feedback button now appears in three locations:
  - Header (blue button, most prominent)
  - Settings page (dedicated feedback section)
  - Main form footer (quick link)
- Integrated Google Forms for collecting user feedback, bug reports, and feature requests
- Analytics tracking for feedback button clicks

### Improved
- Enhanced user engagement with easier feedback collection
- Better visibility for feedback options
- Streamlined feedback submission process

### Fixed
- Analytics tracking error when opening feedback form

---

## [1.0.0] - 2025-01-XX

### Initial Release
- GST-compliant invoice generation
- Professional PDF export
- Business logo support
- Client templates (save and reuse clients)
- Saved items library
- Invoice history
- Accordion sections for better UX
- Auto-save functionality
- Keyboard shortcuts
- CGST/SGST/IGST automatic calculation
- Real-time validation
- Unlimited invoices (100% free)
